    <strong> tag is used in HTML5 whereas <b> tag is used in HTML for displaying a text in bold.
 
    <em> tag or the <i> tag is used in HTML for displaying a text in Italic.
    
    <u> tag or the <ins> tag usage results in text being underlined.
    
    Anything that appears in a <strike> tag is displayed with strikethrough, which is a thin line through the text.
